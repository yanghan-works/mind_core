# PyPI

**Python Package Index** - Python 官方包仓库，是 [[Python 包]] 的中央存储库。

## 基本信息

- **官网**: https://pypi.org
- **包数量**: 40万+ 个包
- **用途**: 存储和分发 Python 包

## 访问方式

- [[pip]] 默认从 PyPI 下载包
- [[conda]] 可以配置使用 PyPI 源
- 网页浏览和搜索包

## 镜像源

由于网络原因，国内用户常使用 [[PyPI 镜像源]]：
