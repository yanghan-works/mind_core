# 项目管理

Python 项目的组织和管理最佳实践。

## 项目结构

```
project/
├── README.md
├── requirements.txt      # [[pip]] 依赖
├── environment.yml       # [[conda]] 环境
├── .gitignore
├── src/
│   └── __init__.py
├── tests/
├── docs/
└── data/
```

## 环境管理策略

- **一项目一环境** - 使用 [[虚拟环境]] 或 [[conda 环境管理]]
- **环境文档化** - 维护 [[requirements.txt]] 或 [[environment.yml]]
- **版本锁定** - 应用 [[包版本控制]] 策略

## 依赖管理

- [[依赖管理]] - 核心策略
- 区分开发、测试、生产依赖
- 定期更新和安全检查
