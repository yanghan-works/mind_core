# Python 包管理器

Python 生态系统中的包管理工具，用于安装、管理和分发 Python 包。

## 主要包管理器

- [[pip]] - Python 标准包管理工具
- [[conda]] - Anaconda 生态系统的包管理器
- pipenv - 高级包管理工具
- poetry - 现代依赖管理工具

## 核心概念

- [[Python 包]] - 可重用的代码模块
- [[PyPI]] - Python 包索引仓库
- [[虚拟环境]] - 隔离的 Python 运行环境
- [[依赖管理]] - 项目依赖的版本控制
